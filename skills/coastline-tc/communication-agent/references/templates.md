# Communication Templates

Loaded by Communication Agent only when a draft needs a small fix or fill-in. These are canonical templates for routine sends. Use them to verify a draft, fill a missing piece, or replace a draft that's broken in a way the agent can fix without rewriting.

If a template doesn't exist for a scenario, do not improvise. Bounce.

**Universal rules:** No em dashes. No advising. Use "typically" / "in the past I've seen" framing where context-sharing helps. Recap, don't explain.

## 12. Mirror Docs Decision Question to Buyer (SubTo / Hybrid only)

**When:** Day 1 of docs phase on a SubTo or Hybrid deal. `mirror_docs_decision: pending`.
**Recipient:** Client (buyer).

> Subject: [Property Address], quick question on mirror docs
>
> Hi [Client First Name],
>
> Now that we're moving into docs phase on [Property Address], wanted to raise an optional step that comes up on most SubTo deals: mirror docs.
>
> The short version: a mirror promissory note + deed of trust drafted to protect the seller in case payments on the underlying mortgage are ever missed. It puts a lien on the property in the seller's favor that mirrors the underlying loan. Without it, the deal is what's sometimes called a "naked SubTo".
>
> Typically I recommend doing this. Two reasons. Ethically, it protects the seller (which is the right thing to do). And in the long run, it also protects you. If the seller ever decides down the road to claim they were taken advantage of, mirror docs are evidence of good faith and that you were looking out for them.
>
> Trade-offs to know about:
>
> 1. Extra cost. The docs are typically drafted by an attorney (or there's an online service called bundl that handles this in many states for less). I can help coordinate either route.
> 2. Adds an additional lien on the home (the whole point, but worth flagging).
> 3. Lien release at payoff is more complex. If the underlying mortgage is paid off in 20 years, we need the seller's signature to release the lien. Sometimes we get a pre-signed release upfront; sometimes we plan for future coordination. I'll work with the title company on the right setup if you decide to move forward.
>
> Let me know how you'd like to proceed. If yes, I'll start coordinating the drafter and we'll fold this into the deal flow. If no, I'll log the decision and we move on.
>
> [Tiffany / Coastline TC]

**Recipient:** Client (buyer).

> Subject: [Property Address], quick question on how you're taking title

> Hi [Client First Name],

> Now that we're rolling on [Property Address], wanted to confirm how you're planning to take title. This determines a few things downstream so good to lock in early.
>
> Two common options:
>
> 1. LLC (your existing entity, or a new one)
> 2. Personally (less common but happens)
>
> Typically I see investors go with an LLC for the entity protection. Let me know which one fits and I'll set up the rest of the file accordingly. If you have something else in mind, just say so and we can talk through it.
>
> [Tiffany / Coastline TC]
