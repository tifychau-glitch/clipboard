# Daily Client Update, Templates and Tone

Loaded by Document Processor at Step 5 of the daily loop. One update per active deal, every day, no exceptions.

## Tone

- Warm, professional, confident.
- Never vague ("things are progressing").
- Never dramatic ("urgent issue!!").
- Calm, specific, forward-looking.
- The client hired Tiffany. The update should reinforce that they made a good choice.
- Use "typically," "in the past I've seen," "I've had investors do" when adding context. Share experience, do not advise.
- Never explain documents or transaction structure. Route deal-structure questions back to the buyer (the client themselves on their own deal, but if the question is about their structure with the seller, that's their conversation).
- No em dashes. Use periods, commas, colons, or parentheses.

## Subject Line

Always:
```
[Property Address], Daily Update [Date]
```

Example:
```
22017 1st PL W Bothell WA, Daily Update Apr 24
```

## Body Structure

Four sections. Always in this order. Skip a section only if it would be empty AND would not provide reassurance.

### STATUS
One sentence describing where the deal is. Plain English, not phase codes.

- "We're in the document collection phase, on track for the [DATE] close."
- "Title commitment came in today and is being reviewed."
- "Final closing docs are scheduled for tomorrow."

### TODAY
What happened today, or what was actively worked. If nothing happened, say so honestly.

- "Mortgage statement received from the seller. Insurance declaration page still outstanding."
- "Quiet day, no new docs in. I've checked in with [escrow / agent] on outstanding items."
- "Title officer confirmed the title search is in progress, ETA on commitment is [DATE]."

### WAITING ON
Specific items, specific people. No vague "various documents."

- "Waiting on: SubTo Addendum (from listing agent), Insurance Declaration (from seller)."
- "Waiting on: Title commitment (First American, ETA Friday)."

### NEXT
What happens next, with a date if known. Forward motion. Never end on a stall.

- "Next: I expect the title commitment by Friday. I'll send it to you for review as soon as it lands."
- "Next: Once the SubTo Addendum is signed, we move to closing prep. I'm staying on the listing agent every few days until it's signed."

### ACTION NEEDED (only when applicable)

Add this only when the client themselves needs to do something. Make it explicit.

- "Action needed: Please send wire instructions for closing. I'll verify them with you verbally before sending to escrow."
- "Action needed: Decide between [option A] and [option B] by [date] so I can move escrow forward."

If COE is within 7 days and client has any open action item, this section is required.

## Quiet Day Template

For days where nothing actually happened. Still send the update.

> Subject: 22017 1st PL W Bothell WA, Daily Update Apr 24
>
> STATUS
> We're in the document collection phase, on track for the Nov 29 close.
>
> TODAY
> No new docs received today. I followed up with the listing agent on the SubTo Addendum and insurance declaration.
>
> WAITING ON
> SubTo Addendum (listing agent), Insurance Declaration (seller).
>
> NEXT
> Following up again in a few days if nothing comes in. I'll let you know the moment anything changes.
>
> [Tiffany / Coastline TC]

## Formatting Rules

- Plain text. No emoji. No exclamation points unless something genuinely deserves it (closed deal, big milestone).
- Maximum 8 lines of body, including section headers.
- Section labels in CAPS. Sub-content in normal sentence case.
- Sign-off: `[Tiffany / Coastline TC]`, single line, no signature image.
- One blank line between sections.
- Never use the word "issue" or "problem". Use "open item" or just describe it.
- Never apologize for routine delays. If something is genuinely behind because of TC, name it and offer the fix.
- No em dashes. Use periods, commas, colons, or parentheses.

## Things to Never Say

- "Things are moving along."
- "We're working on it."
- "Should be soon."
- "Hopefully by Friday."
- "There's been a slight delay."
- Any sentence that doesn't end with a date or a specific next step.
- Anything that explains how a SubTo / Wrap / Novation / Morby deal works mechanically. That's the buyer's conversation with the seller, not ours.
