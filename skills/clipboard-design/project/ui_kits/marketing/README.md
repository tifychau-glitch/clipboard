# Clipboard Marketing UI Kit

Click-through recreation of the Clipboard marketing site.

## Files
- `index.html` — entry point. Stacks Hero → LogoWall → Features → Pricing → CTA band → Footer.
- `components.jsx` — `Nav`, `Hero`, `LogoWall`, `Features`, `Pricing`, `CTABand`, `Footer`
- `../shared.jsx` — shared logo / icon / pill / button primitives

## Sections
- **Hero** — Navy slab with grid + blurred color blobs, two-column layout with product preview card
- **Logo wall** — Social proof strip of 6 customer logos as wordmarks
- **Features** — Three feature cards, one per accent color (yellow/contracts, violet/agents, teal/live)
- **Pricing** — Three tiers with featured center tier in navy
- **CTA band** — Navy with violet glow, yellow CTA
- **Footer** — Dark slab, 4 columns, stripe motif bottom-right
